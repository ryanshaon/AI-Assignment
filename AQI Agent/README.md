# AQI Simple Reflex Agent

## Description
This project implements a Simple Reflex Agent that reads environmental sensor data from a CSV file and computes AQI.

## Agent Type
Simple Reflex Agent:
- Reacts only to current percept
- No memory
- No internal model

## Environment Type
- Fully Observable
- Deterministic
- Episodic
- Static

## How to Run
python agent.py
